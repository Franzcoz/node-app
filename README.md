# App v1.0.0 — Node Express WebApp

Aplicación web base construida con **Node y Express**.

- - -
## Convenciones

La aplicación se ejecuta mediante un archivo principal llamado app.js. Se
decidió ese nombre para evitar confusiones con el archivo index.html.

- - -
## Instalación

Node.js necesita estar instalado en el sistema.

Descargue la carpeta de proyecto o clone el repositorio, y en el directorio principal instale las dependencias ejecutando:

```npm install express```

- - -
## Ejecución del Proyecto

Para ejecutar el proyecto se definieron los siguientes scripts:

- `npm run start` \--> Ejecutar en modo producción
- `npm run dev` \--> Ejecutar en modo desarrollo

Si desea utilizar la aplicación ejecute el primero, y si desea modificarla es recomendable utilizar el segundo script.
